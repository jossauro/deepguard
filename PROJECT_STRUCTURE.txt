KEY FEATURES IMPLEMENTED
========================

Forensic Techniques:
  1. Error Level Analysis (ELA)         - Compression inconsistency detection
  2. Noise Pattern Analysis             - High-pass filtering + block-wise statistics
  3. Metadata Forensics                 - EXIF, ICC profile, software detection
  4. Copy-Move Detection                - DCT-based block matching for cloning
  5. Face Consistency Check             - Eye reflections, skin texture, lighting analysis

CLI Commands:
  - deepguard analyze <file>            - Full forensic analysis with HTML report
  - deepguard batch <folder>            - Batch process images to JSON
  - deepguard report <file>             - Generate forensic report
  - deepguard metadata <file>           - Extract EXIF/metadata only
  - deepguard compare <file1> <file2>   - Compare two images

Output Formats:
  - HTML Reports                        - Beautiful dark-themed, self-contained reports
  - JSON Exports                        - Machine-readable results for automation
  - Console Output                      - Rich terminal tables and progress indicators

Supported Formats:
  JPEG, PNG, TIFF, BMP, WebP, PDF (first page)

TECHNICAL ARCHITECTURE
======================

1. Models (models.py)
   - Verdict enum: AUTHENTIC, SUSPICIOUS, MANIPULATED
   - ForensicReport: Complete analysis result
   - TechniqueResult: Individual technique output
   - MetadataInfo: Extracted metadata structure

2. Analyzer (analyzer.py)
   - ForensicAnalyzer class orchestrates all techniques
   - Single file analysis: analyze()
   - Batch processing: batch_analyze()
   - Automatic verdict calculation

3. Techniques (techniques/)
   - Each technique is an independent class
   - Consistent API: analyze(image) -> (TechniqueResult, heatmap)
   - Confidence scores and verdicts
   - Detailed results and statistics

4. CLI (cli.py)
   - Click framework for commands
   - Rich library for beautiful terminal output
   - Progress bars and spinners
   - HTML auto-opening in browser

5. Reporting (report.py)
   - Jinja2 templates for HTML generation
   - Self-contained HTML (inline CSS, base64 images)
   - JSON export capability
   - Professional styling (#0a0a0f dark theme, #d4a574 gold accents)

6. Utilities (utils.py)
   - Image loading with format validation
   - NumPy to base64 PNG conversion
   - Colormap application
   - Array normalization

HOW TO USE
==========

Installation:
  $ pip install -e .

Single Image Analysis:
  $ deepguard analyze suspicious_photo.jpg

Batch Analysis:
  $ deepguard batch ./evidence_folder --recursive

Extract Metadata Only:
  $ deepguard metadata photo.jpg --format json

Compare Two Images:
  $ deepguard compare original.jpg edited.jpg

Generate Report Only:
  $ deepguard report photo.jpg --output my_report.html


TESTING
=======

Run all tests:
  $ pytest tests/ -v

Run with coverage:
  $ pytest tests/ --cov=src/deepguard

Specific test file:
  $ pytest tests/test_ela.py -v
